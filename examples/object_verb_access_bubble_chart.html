Object-Verb access bubble chart

Bubble chart similar to the provided GI default, but with modifications to allow for larger bubbles and overlaps as well as a different color scale.

  • Report category: Database activity

  • Required report headers:

    • Primary category: Date (Period start (local time))
    • Secondary category: DB user
    • Tertiary category: Verb
    • Primary value: Sum (Total count)
